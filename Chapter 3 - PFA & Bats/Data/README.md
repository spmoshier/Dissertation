# What factors affect the evolution of organismal traits in bats? A case study using phylogenetic factor analysis

Dataset DOI: [10.5061/dryad.stqjq2ch2](https://doi.org/10.5061/dryad.stqjq2ch2)

## Description of the data and file structure

This dataset contains empirical trait data for two groups of bats (*Myotis* and Phyllostomidae) and the code and files necessary to simulate trait data under different scenarios.

#### Phylogeny Files

* Myotis_phylogeny.txt: We trimmed a UCE phylogeny for the genus Myotis (calibrated with fossil data from Gunnell et al (2017)) from Morales et al (2019). There are 33 species in this phylogeny.
* Phyllostomidae_phylogeny.txt: We trimmed a 31-gene mammalian phylogeny published by Upham et al (2019). There are 104 species in this phylogeny.

## Code/software

We used R, Julia, Perl, and command-line batch scripting for this work. Simulating trait data and phylogenies was done using R version 4.4.1, phylogenetic factor analysis was performed using Julia version 1.8.5. Perl and command line scripts were used to perform batch operations on the College of Arts and Sciences Unity Cluster at The Ohio State University. Complete code can be found at [https://github.com/spmoshier/BatPhylogeneticFactorAnalysis.git](https://github.com/spmoshier/BatPhylogeneticFactorAnalysis.git).
